Project Details

Heart Strawng is a health monitoring application designed by HealthyBois. This application will monitor water intake and heart rate through specialized connnected sensors. You can access your data on the mobile application to track various progress goals, such as weight and workouts.

Learn More

Challenges

Overall this project will have many challenges related to both software and hardware. The biggest software challenge will be configuring the communication protocol between the app and sensors. Then, the biggest hardware challenge is going to be making the monitoring straw sensor small, portable, and figuring out a way to keep it charged.
